# Resources

Purpose

Resources are practical tools that help people take immediate action.

They should be:
- free
- genuinely useful
- beautifully designed
- evergreen
- connected to books and articles

Quality Standard

Every resource should be valuable enough that someone would happily pay for it.

Our philosophy is to give it away for free to build trust.

Workflow

1. ChatGPT creates the framework and content.
2. Codex creates the website integration.
3. Resource is added to the Resources page.
4. Premium PDF is created.
5. Future editable versions may be created.
