# Design System

## Visual Direction

Premium editorial publishing platform.

The site should feel:
- elegant
- calm
- intelligent
- practical
- modern
- timeless

Reference feeling:
- Apple
- Stripe Press
- Linear
- James Clear
- high-end book publishers

Never use generic startup design.

## Typography

Use one consistent typography system across the site.

### Display Title

Used for hero headlines.

Large, editorial serif, bold.

### Section Title

Used for major section headings.

Editorial serif, strong weight.

### Eyebrow

Small uppercase labels.

Gold.

Wide letter spacing.

### Body Large

Used for hero descriptions and important paragraph text.

Readable, calm, generous line height.

### Body

Used for normal paragraphs.

Clear, readable, not too small.

---

## Spacing

Use an 8px spacing system.

Common values:
- 8px
- 16px
- 24px
- 32px
- 48px
- 64px
- 96px
- 120px

Prefer generous spacing over cramped layouts.

---

## Buttons

Buttons should feel premium and tactile.

Primary:
- gold background
- dark text
- rounded pill shape
- subtle hover lift

Secondary:
- transparent or dark background
- gold or cream border
- cream text
- subtle hover state

Avoid loud colours.

---

## Cards

Cards should:
- have soft borders
- use subtle shadows
- lift slightly on hover
- never feel busy
- keep text readable

---

## Animations

Animations should be subtle.

Use:
- opacity
- translateY
- scale very lightly
- 200ms to 350ms duration
- ease-out timing

Never use flashy effects.

---

## Layout

Desktop:
- generous whitespace
- strong grid layout
- maximum readable width

Mobile:
- single column
- large tap targets
- readable text
- no horizontal scrolling

---

## Accessibility

Every page should:
- use semantic HTML
- have readable contrast
- support keyboard focus
- avoid tiny text
- avoid motion-heavy interactions

---

## Rule

If a page looks visually inconsistent with the homepage, adjust it to match the shared design system instead of creating new styles.
