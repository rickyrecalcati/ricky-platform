# Ricky Recalcati Writing Guide

## Purpose

This document defines the writing style used across the Ricky Recalcati Platform.

Every piece of writing should sound like it was written by the same person.

Never sound like generic AI.

Never use marketing clichés.

Never exaggerate.

---

# Tone

Confident.

Calm.

Thoughtful.

Practical.

Intelligent.

Optimistic.

Never arrogant.

Never salesy.

---

# Style

Write clearly.

Prefer short sentences.

Avoid unnecessary adjectives.

Every paragraph should teach something.

If something can be explained simply, explain it simply.

---

# Headlines

Strong.

Clear.

Curiosity-driven.

Avoid clickbait.

Examples:

Build a business that works without you.

Why most systems fail.

Better decisions compound.

The system is the service.

---

# Business Writing

Always practical.

Frameworks before theory.

Real examples.

Actionable advice.

Respect the reader's intelligence.

---

# Editorial Style

Calm.

Specific.

Never inflated.

Create value through clear examples, useful questions and disciplined thinking.

---

# Calls to Action

Simple.

Examples:

Buy on Amazon

Read the first chapter

Download the checklist

Join the newsletter

Never:

Buy Now!!

Limited Time!!

Don't Miss Out!!

---

# Philosophy

The platform exists to help people:

Build better businesses.

Make smarter decisions.

Create more freedom.

Use practical experience well.

Everything should reinforce that mission.
